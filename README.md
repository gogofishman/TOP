## 说明

以下是本科技会怎么报，以及需要使用哪些打法等

（不需要特殊说明的tts就不在下面拿出来强调了）





### P1

- **踩塔**：直接告诉你踩哪个塔（比如”踩左塔“）

- **拉线**：不会告诉你拉到哪里去，但是会告诉你优先级大还是小
- **拉线标记**：会给当前拉线的两人标记锁链1、2，优先级大的标1，方便看

- **鲶鱼精播报**：（这个一般拿来复盘更多一点）

![](spl/img/1.png)

- **全能之主**：告诉你去上半场还是下半场
